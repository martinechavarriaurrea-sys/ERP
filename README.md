# ERP Personal

Aplicación ERP personal en un solo archivo HTML (`index.html`), sin backend y con persistencia en `localStorage`.

## Ejecutar local

Abre `index.html` en el navegador.

## Deploy en Vercel

1. Importa este repositorio en Vercel.
2. Framework Preset: `Other`.
3. Root Directory: `.`
4. Build Command: vacío.
5. Output Directory: vacío.

Vercel servirá `index.html` como sitio estático.

Integración GitHub-Vercel activa en la rama `main`.
